Help and Instructions

Website Navigation

Use the navigation bar at the top of every page to move through the website:

  • Home: The main landing page with general information and announcements.
  • Gallery: View collections of images related to university events and activities.
  • Booking: Fill out the guest booking form to register your details and interests.
  • About: Learn more about Jamhuuriya University, its history, and mission.
  • Help: (This page) Find instructions and tips for using the website.

Using Forms

Forms allow you to submit information to the university. For example, the Booking page contains a form to register as a guest.

  • Input Fields: Fill out required fields marked with an asterisk (*) or the required attribute.
  • Dropdown Menus: Select an option from the dropdown list (e.g., your major).
  • Radio Buttons: Choose one option only, such as your gender.
  • Checkboxes: Select one or more hobbies or interests.
  • Textarea: Write any additional comments or information.
  • Submit: Click the "Submit Booking" button after filling out the form. If any required field is empty or invalid, the browser will prompt you to correct it.

Interacting with Images

Images are displayed primarily on the Gallery page to showcase university life.

  • Hover over images to see their description (via the alt text).
  • Click on images to view them in larger size or in a slideshow if available.
  • Images help you get a visual impression of campus events, buildings, and activities.

Using Tables

Tables are used to organize data clearly and efficiently.

  • Look for tables on pages like Gallery (for image info) or other data-driven pages.
  • Headers in the table help you understand the meaning of each column.
  • Use the scrollbar if the table is wider than your screen.
  • Tables make it easier to compare information such as schedules or event details.
